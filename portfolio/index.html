portefolio van wassim

Volgende keer wil ik toch wel wat sneller werken zodat ik meer vooruitgang zie. Ik wil ook meer opletten in de les omdat ik meestal te snel aan het werk wou zijn waardoor ik bijvoorbeeld een belangrijke uitleg miste wat ervoor zorgde dat ik niet echt wist wat ik op de toets moest doen gelukkig heb ik mijn herkansing ervoor gedaan anders was het wel een probleempje geweest. Waar ik vanaf nu voor wil zorgen is dat ik wel wat sneller een vraag stel, want ik ben nogal eigenwijs waardoor ik meestal altijd wel alles zelf wil doen. Tot slot wil ik ervoor gaan zorgen dat ik meer met javascript ga werken want eigenlijk vond ik javascript echt 10 keer niks, tot ik het een beetje begon te begrijpen en eigenlijk erachter kwam dat er best wel veel logica in zit waardoor ik het sneller begrijp.

Ik heb van de afgelopen tijd dat ik aan mijn portfolio heb gewerkt geleerd dat er oneindig veel mogelijkheden zijn om je portfolio te ontwerpen. Ik heb er ook van geleerd dat je portfolio laat zien hoeveel moeite en tijd je erin hebt gestopt. Er zal altijd wel iets zijn wat je eraan wilt toevoegen maar op een gegeven moment zal je toch wel echt door moeten gaan als je je portfolio wilt afmaken. ik heb er ook van geleerd dat het belangrijk is om netjes te coderen want jij zal het mischien wel overzichtelijk genoeg vinden maar als je een leraar om hulp vraagt kan het best wel een grote moeite kosten om iets te vinden wat eigenlijk best wel simpel is.

wat ik nog zou willen leren zijn de volgende dingen. Ik zou graag willen leren hoe ik abstracte live achtergronden kan maken die reactive zijn. Wat ik ook wil leren is hoe ik mijn buttons ook echt iets kan laten versturen bijvoorbeeld een mailtje. Ook zou ik heel graag willen leren in welke situaties je welke taal of code moet gebruiken voor de meest mooie resultaten. Tot slot zou ik zo veel mogelijk talen willen leren zodat ik alles wat ik wil maken ook kan omzetten in code bijvoorbeeld de manier hoe je een app moet bouwen of een game.
